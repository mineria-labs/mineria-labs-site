プライバシーポリシー

最終更新日：2026年4月4日

Mineria Labs（以下「当社」）は、鉱物標本コレクション管理アプリ「Mineria」（以下「本アプリ」）における 利用者の個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 収集する情報

本アプリは以下の情報を収集することがあります。

  • ニックネーム・メールアドレス（アカウント登録時）
  • プロフィール画像（任意）
  • 標本データ（名称・産地・写真等、ユーザーが入力した情報）
  • ご利用のデバイス情報・OS種別（クラッシュ解析目的）
  • アプリの利用状況（AI識別の利用回数等）

2. 情報の利用目的

  • 本アプリのサービス提供・機能改善
  • ユーザーサポートへの対応
  • AI識別機能の利用制限管理
  • 課金・サブスクリプションの管理
  • 不正利用の防止

3. 第三者への提供

当社は、以下の場合を除き、収集した個人情報を第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく開示が必要な場合
  • サービス提供に必要な業務委託先への提供（守秘義務を課す）

4. 利用する外部サービス

本アプリは以下の外部サービスを利用します。各サービスのプライバシーポリシーもご確認ください。

  • Firebase（Google LLC）— 認証・データベース・ストレージ
  • Cloudflare R2（Cloudflare, Inc.）— メディアファイル保存
  • Google Gemini API（Google LLC）— AI識別機能

5. 登録情報の保護

登録した標本情報（購入価格・入手情報・説明文メモ等）は、すべてご本人のみが閲覧できます。 第三者に送信・表示されることはなく、サーバー側でもアクセス制御を行っています。 ただし、ユーザーご自身がSNS等の外部サービスでシェアした内容については、本条の対象外となります。

6. データの保管・削除

ユーザーはいつでもアカウントを削除できます。削除時にはFirestoreのユーザーデータおよびストレージ上の 画像ファイルを完全に削除します。ゲスト（匿名）アカウントは、アプリのアンインストールにより 端末上のデータが失われます。

7. セキュリティ

Firestoreのセキュリティルールにより、自分のデータのみ書き込み可能な設計としています。 APIキー等の機密情報はコードに含めず、Firebase Remote Config等で管理しています。

8. お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。 13歳未満のお子様から個人情報を収集していることが判明した場合、速やかに削除します。

9. ポリシーの変更

本ポリシーは予告なく変更されることがあります。 重要な変更がある場合はアプリ内のお知らせ機能を通じてご連絡します。

10. お問い合わせ

プライバシーに関するご質問は、アプリ内の「お問い合わせ」フォームよりご連絡ください。

メール：support@mineria-labs.com

Privacy Policy

Last updated: April 4, 2026

Mineria Labs ("we") operates the mineral specimen collection app "Mineria" ("the App"). This Privacy Policy explains how we handle your personal information.

1. Information We Collect

The App may collect the following information:

  • Nickname and email address (when registering an account)
  • Profile image (optional)
  • Specimen data (name, locality, photos, and other user-entered information)
  • Device information and OS type (for crash analysis)
  • App usage data (e.g., number of AI identifications used)

2. How We Use Your Information

  • Providing and improving the App
  • Responding to user support requests
  • Managing AI identification usage limits
  • Managing billing and subscriptions
  • Preventing unauthorized use

3. Sharing with Third Parties

We do not share collected personal information with third parties except in the following cases:

  • With your consent
  • When required by law
  • With service providers necessary for operating the App (subject to confidentiality obligations)

4. Third-Party Services

The App uses the following external services. Please also review their respective privacy policies.

  • Firebase (Google LLC) — Authentication, database, storage
  • Cloudflare R2 (Cloudflare, Inc.) — Media file storage
  • Google Gemini API (Google LLC) — AI identification feature

5. Protection of Registered Information

All registered specimen information (including purchase price, acquisition info, and notes) is accessible only to the user who registered it. It is never transmitted to or displayed for third parties, and access is controlled on the server side. Note that content voluntarily shared by the user via social media or other external services is outside the scope of this section.

6. Data Retention and Deletion

You can delete your account at any time. Upon deletion, all Firestore user data and stored image files are permanently removed. Guest (anonymous) account data is lost if the app is uninstalled.

7. Security

Firestore security rules ensure that only your own data can be written. Sensitive information such as API keys is not included in the code and is managed via Firebase Remote Config.

8. Children's Privacy

The App is not intended for children under 13. If we discover that personal information has been collected from a child under 13, we will promptly delete it.

9. Changes to This Policy

This Policy may be updated without prior notice. We will notify you of significant changes through the in-app announcement feature.

10. Contact Us

For privacy-related questions, please use the "Contact Us" form in the App.

Email: support@mineria-labs.com
